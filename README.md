## 琼斯医生

💊 一个能够美化网页中文排版（包括中英文混排）的扩展

## 特性

- 遵循 [w3c/clreq](https://github.com/w3c/clreq) 和其他中文排版最佳实践，格式化你正在浏览的页面
- 所有格式化功能均独立可配置，你可以按照自己的喜好打开/关闭
- 支持在页面打开后自动格式化，同时支持配置自动格式化的黑名单
